Main Projects

In Progress: World Bank GDP Data Pipeline (ETL, Data Transformation & Analytics)

Designed and implemented a scalable end-to-end ETL pipeline for extracting, transforming, and analyzing World Bank GDP data at scale. Built modular extraction components with paginated API calls to the World Bank API, implementing intelligent error handling and logging for production reliability. Developed a multi-layer cloud architecture leveraging AWS S3 with Hive-style partitioning (raw/processed/analytics), organizing data for scalability and downstream analytics. Implemented PySpark-driven transformation workflows for large-scale data cleaning, type casting, null handling, and aggregation across millions of records. Created analytics layer with grouped aggregations, statistical computations (average GDP per country), and production-grade structured logging for debugging and monitoring. Orchestrating scheduled pipeline runs using Apache Airflow DAGs, enabling automated daily extraction, transformation, and analytics aggregation with dependency management and monitoring. Focus: building reliable, scalable ETL systems that move from raw data ingestion through transformation to actionable insights, with emphasis on cloud architecture, distributed computing, and workflow orchestration.

Customer Churn Data Pipeline (ETL, Feature Engineering & ML Deployment)

Designed and implemented an end-to-end data pipeline for customer churn prediction, covering data ingestion, preprocessing, feature engineering, model training, and deployment. Built reusable ETL components for data cleaning and transformation, including handling missing values, feature binning, and statistical aggregation. Integrated machine learning models into a production-ready workflow, including model evaluation, threshold optimization (profit-based), and prediction services. Deployed the system using Render, with AWS services (S3, Cognito, WAF) for storage, authentication, and security. Focus: building reliable, modular, and scalable data pipelines for ML-driven decision systems.

NLP Pipeline for Sentiment Analysis on Product Reviews

Implemented neural network models for sentiment classification and integrated them into an interactive Streamlit dashboard. Designed the pipeline to support dynamic filtering by product, category, and time, enabling real-time insight exploration. Impact: Demonstrates ability to design production-ready data pipelines that support business decision-making and ML-driven optimization.
